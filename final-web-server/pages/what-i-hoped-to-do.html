What I hoped to do

What was my goal with this project?

My goal was to create a simple webserver, that could respond to HTTP requests, and generally show a website. I wanted to write it in a language called Rust.

Goals

  1. Be able to respond to HTTP requests
    • Read in a tcp stream
    • Parse HTTP header
    • Send data back to the TCP stream
  2. Be able to get arbritrary files
    • Parse what file the user is looking for
    • Check if that file exists
    • Make sure the user is allowed to see that file
    • Read in the file and send it to the user
  3. Working 404 page and some other error handleing
    • If a file that the user is looking for does not exist, show them the 404 page instead
    • Respond with a 404 error code
    • If the client sends anything I am unable to parse through, then respond with a 500 error
